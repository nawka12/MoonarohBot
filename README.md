# MoonarohBot
Discord music bot using discord-player and @distube/ytdl-core.

Dependencies:
- Node.js 20 or up
- NPM

How to setup:
```bash
git clone https://github.com/nawka12/MoonarohBot
cd MoonarohBot
npm install
```
Put your Discord Bot token inside `.env`

Then:
```bash
node index.js
# or do 'node install -g pm2' then
pm2 start index.js
```
